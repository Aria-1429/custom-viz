# Custom Viz Radial Bar（ラジアルバーチャート）

![表示例](examples/example.png)

Splunk Dashboard Studio 向けカスタムビジュアライゼーション。
カテゴリを **放射状のバー（環状セクター）** で表現し、**値をバーの外側への伸び**として示す。
中央に合計を大きく表示する、モダンでダッシュボード映えするチャート。

## 特徴

- **放射状カラムチャート**：等角のくさびを円状に並べ、値が大きいほど外側に伸びる。
  標準の棒グラフ／円グラフでは出せない「レーダー状の一覧性 + 中央 KPI」を1枚で表現。
- **データドリブン**：サーチ結果（SPL）に応じて描画が変わる。
  第1列＝カテゴリ、第2列以降＝数値。値の大きい順に上位 `maxBars` 本を描画。
- **値ベースの色**：低→（中）→高 の3色補間でバー色を自動決定（既定 ON）。
  値ベースを OFF にすればカテゴリ別のパレット色になる。
- **フィールド選択**：編集画面でカテゴリ列・値列をドロップダウンから選べる。
- **ホバー連動**：バー／ラベルにホバーすると中央表示がそのカテゴリの値・ラベルに切り替わる。
- **ライト／ダーク両対応・オートフィット**：パネルサイズに合わせてラベル余白・文字サイズを調整。

## データ仕様

| 列 | 役割 | 例 |
|----|------|-----|
| 1列目 | カテゴリ（ラベル） | `DC-SG-G02` |
| 2列目 | 数値（バーの長さ） | `9800` |

- 編集画面の「データ設定」でカテゴリ列・値列を明示的に選択可能（未指定なら 1列目＝カテゴリ、
  最初の数値列＝値に自動フォールバック）。
- 負値・非数値・空ラベルの行は除外。マルチバリューセルは平行展開で救済。

## サンプル SPL

```spl
| makeresults format=csv data="product_group,revenue
DC-SG-G02,9800
MB-AG-G07,8200
WC-SH-G05,7100
DB-SG-G01,6400
FI-AG-G08,5600
FS-SG-G03,4900
SC-MG-G10,4100
MB-MG-G09,3300
CU-PG-G06,2600
MB-AG-T01,1300"
```

中央合計に `$` を付けたい場合は編集画面「ラベルと中央 > 中央の値に $ を付ける」を ON にする。

## 主なオプション（編集画面）

- **データ設定**：カテゴリのフィールド／値のフィールド
- **リング**：最大バー数、バー間の隙間（度）、内側の半径（％）、バー端の丸み（px）、
  背景トラック表示、発光エフェクトと強さ
- **ラベルと中央**：外周ラベル表示、バー上の値表示、中央合計表示、中央値への `$` 付与
- **色（値ベース）**：値に応じて色を変える／中間色を使う／高↔低を反転／低・中・高の色
- **色（カテゴリ別）**：値ベースが OFF のときのバー 1〜6 の色

## 開発

```bash
yarn install
yarn build      # dist/custom_viz_radial_bar/visualization.js を生成
yarn verify     # happy-dom でローカル検証（Splunk 実機不要）
yarn package    # dist/custom_viz_radial_bar-<ver>-<hash>.spl を生成
```

## デプロイ（アンインストール・再起動なし）

1. `yarn build && yarn package` で `.spl` を生成。
2. Splunk Web「Install app from file」で **"Upgrade app"（上書き）にチェック**してアップロード。
3. ブラウザで `https://<host>:8000/en-US/_bump` を開き **Bump version**。
4. ブラウザをハードリロード（Ctrl+Shift+R）。

> バージョンを上げるたびに `package.json` と `package/app/app.conf` の `version` を同期すること。
