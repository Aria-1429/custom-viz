# Custom Viz KPI Tile

![表示例](examples/example.png)

Splunk Dashboard Studio 向けのカスタムビジュアライゼーション。
SOC ダッシュボードでよく見る「タイトル＋大きな数値＋前日比＋ミニ棒グラフ（スパークライン）＋アイコンバッジ」を、アクセントカラーで統一されたネオン調の 1 枚タイルとして表示する。複数パネルを横に並べて KPI 列を構成する使い方を想定。

## 特徴

- **大数値**: サーチ結果の最終行の値をカンマ区切り（または `1.5M` 省略表記）で大きく表示。カウントアップアニメーション付き
- **前日比**: 最終行 − 直前行 を `↑ 15 (前日比)` の形式で表示。％表示・増減での色分け（増=緑/減=赤、反転可）に対応
- **スパークライン**: 全行の値をミニ棒グラフで表示(本数制限可、新しいほど明るい)
- **アイコンバッジ**: 12種（シールド/家/警告/稲妻/ユーザー/監視/地球/錠前/バグ/サーバー/ベル/波形）。**ダッシュボード編集モード中にタイル上のアイコンをクリックするとピッカーが開き、その場で選択できる**（編集パネルの「アイコン番号」でも指定可）
- アクセントカラー 1 色からタイル全体（グラデ背景・縁・発光・文字・棒）を自動配色。ライト/ダークテーマ両対応
- コンテナ実寸へ自動フィット。小さいパネルではスパークライン → 前日比 → タイトルの順に段階退避

## データ仕様

1 行 = 時系列の 1 ポイント（行順が古 → 新）。

| 列 | 意味 | 既定 |
|---|---|---|
| ラベル（時刻）列 | スパークラインのツールチップに使用 | 第1列 |
| 値列 | 数値。最終行が大数値、直前行との差が前日比 | 最終列 |

- フィールドは編集パネルの「データフィールド」（columnSelector）で選択可
- タイトルは値フィールド名を表示（SPL の `rename` で日本語化するのが手軽）。`titleText` オプション（ソース JSON 編集）でも上書き可
- 「前日比」の注記文字列は `deltaLabel` オプション（ソース JSON 編集）で変更可（例: 前週比）
- 1 列だけの結果は「値のみの系列」として扱う

## サンプル SPL

```spl
| makeresults count=30
| streamstats count as day
| eval _time = relative_time(now(), "-" . (30 - day) . "d@d")
| eval 重大アラート = 10 + (day * 7) % 17 + if(day = 30, 6, 0)
| table _time, 重大アラート
```

実データ例（日次のアラート件数）:

```spl
index=security sourcetype=alerts severity=critical earliest=-30d@d
| timechart span=1d count as 重大アラート
```

## オプション（ダッシュボード編集画面）

- **データフィールド**: ラベル（時刻）フィールド / 値フィールド
- **アイコン**: 表示 ON/OFF、アイコン番号 1〜12（編集モード中はタイル上のアイコンをクリックしても選択可）
- **色・発光**: アクセントカラー、発光（グロー）
- **表示**: タイトル / 前日比 / ％表示 / 増減色分け（＋反転） / スパークライン（＋本数）
- **数値**: 小数桁、省略表記（1.5M）、カウントアップアニメーション
- **デバッグ**: オプションのデバッグ表示

## 開発

```bash
yarn install
yarn build     # dist/custom_viz_kpi_tile/visualization.js
yarn verify    # happy-dom によるローカル検証（Splunk 実機なし）
yarn package   # dist/custom_viz_kpi_tile-<ver>-<hash>.spl
```

## デプロイ

1. `yarn package` で生成された `.spl` を Splunk Web「App の管理 > ファイルから App をインストール」でアップロード（更新時は「App のアップグレード」にチェック）
2. `https://<host>:8000/en-US/_bump` で Bump version（再起動不要）
3. ブラウザをハードリロード（Ctrl+Shift+R）

Dashboard Studio のソース JSON では `"type": "custom_viz_kpi_tile.custom_viz_kpi_tile"` を指定する。
